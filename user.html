🔐 Supabase অটো-লগিন ইউজার ম্যানেজমেন্ট

ইউজার তৈরি করলেই সরাসরি লগইন হয়ে যাবে — আলাদা লগইন প্রয়োজন নেই

➕ নতুন ইউজার যোগ করুন

📋 সকল ইউজার তালিকা 0

ক্রমিক ইমেইল তৈরির তারিখ ক্রিয়া
ডাটা লোড হচ্ছে...

📌 Supabase Auth টেবিল সেটআপ:

Supabase ড্যাশবোর্ডে SQL Editor এ গিয়ে এই কোড রান করুন:

CREATE TABLE users (
    id UUID PRIMARY KEY,
    email VARCHAR(255) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT NOW()
);

⚠️ গুরুত্বপূর্ণ: Supabase Authentication এ Disable Email Confirmation করে দিন:
→ Authentication → Settings → Confirm email: OFF করুন।
তাহলে ইমেইল কনফার্মেশন ছাড়াই ইউজার তৈরি হবে এবং অটো-লগইন হবে।
